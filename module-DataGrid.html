DataGrid

Data Grid that replaced CustomDataGrid based on MUI's DataGrid. Unfortunately, as nice as MUI is, its DataGrid does not have native (and free) Virtualization, Pagination, and Sorting. With larger databases, as ITAD will most likely grow, all three of these are important for a dynamic and fast website. Therefore, this new Data Grid supports all three quite well. It has the following features:

  • Abstracts away some default properties
  • Height is easily-settable and the grid, plus footer, will take up the whole height
  • Adds a footer with pagination and searching, currently will always exist, but will add a flag to disable footer later
  • Width resizes to fit the columns passed in
